# Lesson Vue ファッションサイト

## 技術詳細

- Vue3 CompositionApi
- TypeScript
- Tailwind CSS 3.47

## 開発環境

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
